# Raspberry-Pi-Smart-Motion-Logger
A Raspberry Pi project that detects motion using a PIR sensor and logs activity with timestamps.

## Features
- Motion detection via PIR sensor
- Event-based logging to CSV
- Clean GPIO setup and cleanup
- Designed for security and automation use cases

## Hardware
- Raspberry Pi
- PIR Motion Sensor

## Applications
- Room activity monitoring
- Smart security systems
- Home automation triggers

## Future Improvements
- Camera capture on motion
- Email or Telegram alerts
- Web dashboard
